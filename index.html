This website is a complete car management platform developed for my company at my time as an Intern, designed to streamline the operations of managing vehicles and employees. It allows administrators to track the total number of cars, monitor their status, and manage employee assignments efficiently.
The platform features two separate interfaces:

Admin Site: Enables administrators to oversee all vehicles, manage employee records, and generate reports on car utilization and staff activity.

User Site: Allows employees to view assigned vehicles, update relevant information, and track their own work schedules.

Built using Angular for a responsive, dynamic front-end and Firebase for authentication and real-time database capabilities, the system ensures secure data handling and seamless performance across all devices.
